# kakeibo 🐷

> Easy household budget management

シンプルで使いやすい家計簿アプリです。単一のHTMLファイルで動作します。

## 使い方

1. `index.html` をブラウザで開くだけで使えます
2. `icon-180.png` は `index.html` と同じフォルダに置いてください

## 機能

- **固定費** 毎月の固定支出を管理。支払い済みはチェックで管理
- **変動費** カテゴリごとに内訳メモを記録。スクロール対応
- **収入** 給与・副業など収入を管理
- **貯金** 積立・貯蓄の目標管理
- **サマリー** 残高・収入・支出・貯金合計をひと目で確認
- **先月比** 前月との収支比較
- **undo/redo** 内訳の操作を元に戻せる
- **削除取り消し** 削除後7秒間は元に戻せる
- **データ保存** localStorageに自動保存

## 技術仕様

- 単一HTMLファイル（外部依存なし、Google Fontsのみ）
- vanilla JS / CSS
- localStorage による永続化
- XSS対策済み（escHtml）
- iOS Safari対応（ズーム防止）
